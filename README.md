# 📊 Finance Analysis Dashboard

## 📌 Project Description

=> This report analyzes **financial performance** across products, countries, and customer segments. It helps businesses monitor key metrics like sales, profit, cost, and gross revenue, enabling better decision-making and performance evaluation.

---

## 📊 Data Source

=> Dataset is sourced directly from **Microsoft Power BI sample datasets**.

---

## 🛠 Tech Stack

=> Power BI – used for  visualization.

---

## 🔧 Tasks Performed

Built an interactive report with the following insights:

     => Financial performance by Product, Country, and Segment
     => Profit and sales distribution across categories
     => Comparison of revenue and cost metrics for better business insights

---

## ✨ Key Features

Cards (used as KPIs):

     => Sum of Sales
     => Sum of Profit
     => Total Units Sold
     => Manufacturing Price
     => Number of Products
     => Sum of Gross Sales
     => Sum of COGS

Cards (used as buttons for navigation):

     => Country
     => Product
     => Segment

Slicers:

    => Country
    => Product
    => Segment

Charts & Visuals:

=> **Product View**

     * Bar Charts: Count of Unit, Sum of Unit Sold, Manufacturing Price, Sum of Gross Sales (by Product)
     * Pie Charts: Sum of Sales and Sum of Profit (by Product)

=> **Country View**

     * Bar Charts: Count of Unit, Sum of Unit Sold, Manufacturing Price, Sum of Gross Sales (by Country)
     * Pie Charts: Sum of Sales and Sum of Profit (by Country)

=> **Segment View**

     * Bar Charts: Count of Unit, Sum of Unit Sold, Manufacturing Price, Sum of Gross Sales (by Segment)
     * Pie Charts: Sum of Sales and Sum of Profit (by Segment)

Navigation:

      => Card-based buttons for switching between Country, Product, and Segment views

---
